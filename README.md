# ruah-blog

마크다운 blog 만들기 프로젝트

## 프로젝트 목표

이 프로젝트는 사용자가 마크다운 파일을 업로드하여 블로그 포스트로 변환할 수 있는 웹 애플리케이션을 만드는 것을 목표로 합니다.

## 기술 스택

- **React**: 사용자 인터페이스를 구축하기 위한 JavaScript 라이브러리
- **GitHub Actions**: 지속적 통합 및 배포(CI/CD)를 위한 자동화 도구

## 기능

- 마크다운 파일 업로드 및 렌더링
- 블로그 포스트 관리
- 자동 배포 및 업데이트

## 시작하기

1. 이 저장소를 클론합니다.
2. 필요한 패키지를 설치합니다.
3. 개발 서버를 시작합니다.

```
npm init
```

```
npm install
```
